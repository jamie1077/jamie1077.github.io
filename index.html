Hi, I'm Jamie. I am a front end web developer based in Manchester.

I am a self taught front end web developer and have several years commercial experience designing and building responsive Umbraco websites. My passion for web development started when I worked as a project manager at a software agency. Working alongside designers and developers to create solutions for clients sparked my interest to learn these skills.

My skills include:

  • Development of websites in Umbraco CMS utilising the Modelsbuilder, Umbraco Forms and third party extensions
  • Implementing responsive web design principles to ensure that website renders well across multiple devices
  • Scoping and costing of projects
  • Creation of project wireframes using Adobe XD
  • Concept and layout web design
  • Use of pre-processors such as SASS and the use of module bundler webpack
  • Working with code versioning tools such as GIT in an environment that has a CI and CD workflow
